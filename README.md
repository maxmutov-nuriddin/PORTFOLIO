# Portfolio Website

Personal portfolio website showcasing projects, skills, and experience as a Frontend Engineer.

## Features

- Modern and clean UI design
- Fully responsive layout
- Project showcase with descriptions and live demos
- Skills and technology stack display
- Professional experience timeline
- Contact form integration
- Smooth animations and transitions
- Optimized performance and SEO

## Tech Stack

- TypeScript
- React
- SCSS
- HTML5

## Installation

```bash
git clone https://github.com/maxmutov-nuriddin/PORTFOLIO.git
cd PORTFOLIO
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build for Production

```bash
npm run build
```

## Project Structure

```
PORTFOLIO/
└── portfolios/
    ├── src/
    │   ├── components/    # React components
    │   ├── styles/        # SCSS stylesheets
    │   ├── assets/        # Images and media
    │   └── App.tsx        # Main application
    ├── public/            # Static files
    └── package.json       # Dependencies
```

## Deployment

The website is deployed on Vercel with automatic CI/CD integration. Any push to the main branch triggers a new deployment.
